Installation
Clone the repository:

git clone https://github.com/parneetws/Predictive-Modeling-Using-Machine-Learning
cd Predictive-Modeling-Using-Machine-Learning
Make sure you have Python installed, and install required dependencies:

pip install -r requirements.txt
Usage
Use this repository to:

Experiment with Machine Learning algorithms.
